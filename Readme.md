# expressTemplate

===

## Papar Information

- Title: `express模板项目`
- Authors: `Jerry`

## Install & Dependence

- node js
- mysql
- npm

## Use

```cmd
npm run dev
npm start
```

两个命令都可以 区别是第一个有热重载

## 操作指南

1. 先装包

```cmd
npm install
```

2. 修改 config 文件夹下的配置信息
3. 终端启动
